<h2 style="color:#ff69b4;">Projetos em destaque</h2>

<b>Gerador de Famílias — The Sims 4 (Python)</b>

Projeto desenvolvido em Python que gera famílias aleatórias para o jogo The Sims 4.

O usuário escolhe a quantidade de Sims (1 a 8) e o programa cria automaticamente cada integrante com:

- gênero e idade  
- raça ou criatura (humano, vampiro, sereia, gato, cavalo etc.)  
- traços de personalidade sorteados  

O projeto utiliza listas, funções e estruturas de repetição para organizar a lógica e permitir expansão futura.

<br>

<b>Criador de Personagens — Baldur’s Gate 3 (Java)</b>

Projeto em Java inspirado no sistema de criação de personagens de Baldur’s Gate 3, com foco em Programação Orientada a Objetos.

O sistema permite:

- criação de personagens com nome, raça e classe  
- definição de atributos principais  
- organização do código usando classes, métodos e arrays

O objetivo é praticar a estruturação de sistemas mais próximos de aplicações reais.
